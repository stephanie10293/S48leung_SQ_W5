# Week 5 Example 3 — 

## Project Title: Mario-Themed Maze Game

## Setup and Interaction Instructions

Controls

W = move up
A = move left
S = move down
D = move right

How to Play
The player starts in the maze and must collect all of the coins. Once all coins are collected, the exit tile will activate. The player must then reach the exit tile to win the game. A Mario-themed win screen will appear once the player completes the level.
